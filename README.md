# HV_test_iseg
Controller for I-seg THQ Power Supply for HV test

Need Pyserial to be installed 

Minimum Python V3 

How to run : 

py run_HV_test.py -v 5 -s 10 -e 100 -p 5

-v:entering voltage for USB interface
-s: Start voltage of test
-e: End Voltage of test
